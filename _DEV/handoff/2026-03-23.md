# 핸드오프 — 2026-03-23

## 오늘 작업 요약

### 1. 프로젝트 구조 변경
- Electron 제거 → 순수 React + Vite 웹앱으로 전환
- Vercel 배포 기반 (커밋+푸시 → 자동 배포)
- 반응형 레이아웃 (min-width: 1200px, 사이드바 310px 고정, 콘텐츠 flex: 1)

### 2. 메모 → 태스크 전환
- DB: `memos` → `tasks`, `memo_details` → `task_items`, `memo_id` → `task_id`
- `tasks` 테이블에 `assignee` (담당자) 컬럼 추가
- 모든 컴포넌트/UI 텍스트 memo → task 통일
- MemoModal → TaskModal, MemoViewModal → TaskViewModal 리네임

### 3. 회원/인증 수정
- `user_profiles`에 `name` 컬럼 추가 (가입 시 저장)
- 초대코드 사용 후 비활성화 (1회용)
- 관리자 승격/해제 UI (SettingsPanel)

### 4. 대시보드 개편
- 프로젝트 진행률, 빠른 태스크 추가 제거
- 통계 카드 + 프로젝트별 칸반 메이슨리 보드
- content-header: 고정 높이/배경/보더 제거 → 콘텐츠 안에 자연스럽게

### 5. 사이드바 정리
- 대시보드 nav 항목 제거 → "프로젝트" 타이틀 클릭 시 대시보드
- 섹션 타이틀 스타일 통일 (14px, 600 weight, 클릭 시 navy+bold)
- 구분선 제거, 업무 일람 메뉴 제거 (프로젝트 내 탭으로 이동)

### 6. 업무추진표 (프로젝트 내 탭)
- ProjectDetail에 **대시보드 | 업무추진표** 탭 추가
- DB 구조: `roadmap_rows` (project_id, major, minor, assignee, output) + `roadmap_cells` (row_id, year, month, content)
- TASK 아코디언 카드로 행 추가 (대분류/소분류/담당자)
- 분기별 페이지네이션 (Q1~Q4, 연도 이동)
- 대분류 rowspan 병합, 소분류 없으면 colspan 통합
- 헤더: "업무" (대분류+소분류 통합) / 담당 / 산출물 / 월
- 월별 셀: 체크리스트 (JSON 저장, 항목 추가/체크/삭제)
- 셀 텍스트 1줄 ellipsis + hover 팝오버
- 모든 필드 클릭 인라인 편집, Shift+Enter 줄바꿈
- 행 순서 변경 (↑↓ 버튼)
